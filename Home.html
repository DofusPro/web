¿Como saber los comandos?

Usando  el comando -help o -ayuda .

Logs

para selecionar logs seria -setlogs y para saber cual canal esta -logs y para welcome y leaver -setwelcome -setleave y para saber el canal -welcome -leave.

Moderacion

Los comandos que necesitas para mantener tu servidor a salvo y fuera de raider warn esto dará una advertencia al usuario warnlevel ver las advertencias de un usuario ban banea a un usuario permanente del servidor kick lo expulsa con oportunidad de volver unban para desbanear a un usuario baneado.
